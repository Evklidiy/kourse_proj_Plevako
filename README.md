### 1. ОПИСАНИЕ ПРОЕКТА
Разработка распределенного приложения на C# для решения систем уравнений методом Гаусса-Джордана. 
Система поддерживает работу с 50 000 неизвестных и распределяет вычисления между узлами по сети (TCP).

---

### 2. СТРУКТУРА РЕШЕНИЯ (ПРОЕКТЫ)

[+] Slau.Common (Class Library)
    -- Models/: Классы данных (MatrixChunk, PivotRow)
    -- Networking/: Сетевой протокол и команды управления
    -- Serialization/: Быстрая передача больших массивов данных

[+] Slau.MathEngine (Class Library)
    -- Algorithms/GaussJordanCore.cs: Математика метода Гаусса-Джордана
    -- Algorithms/RelaxationSolver.cs: Метод релаксации (для сравнения)
    -- Storage/LargeMatrix.cs: Работа с памятью для N=50000

[+] Slau.Master (Windows Application)
    -- Distribution/WorkerManager.cs: Управление списком серверов-вычислителей
    -- Distribution/JobDispatcher.cs: Разрезание матрицы на части
    -- Views/MainWindow.xaml: Графический интерфейс управления и графики

[+] Slau.Worker (Console Application)
    -- Network/WorkerServer.cs: Служба ожидания задач от Мастера
    -- Services/ComputationService.cs: Параллельные вычисления на процессоре

[+] Slau.Tests (Test Project)
    -- Unit/: Тесты точности решения (3x3, 10x10)
    -- Load/: Нагрузочные тесты (проверка памяти при 50 000 уравнений)
