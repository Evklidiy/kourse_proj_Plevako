🖥️ Распределенная система решения СЛАУ методом Гаусса-Джордана
Данный проект является курсовой работой по разработке распределенного приложения для решения систем линейных алгебраических уравнений (СЛАУ) большой размерности.

🚀 Особенности реализации
Масштабируемость: Поддержка систем до 50 000 неизвестных (обработка ~18.6 ГБ данных в распределенном режиме).
Архитектура: Master-Worker (Ведущий-Ведомый) на базе TCP-сокетов.
Сравнение методов: Реализован распределенный метод Гаусса-Джордана и локальный метод релаксации (Якоби) для оценки эффективности.
Оптимизация: Использование double[][] (массивов массивов) для обхода лимита CLR в 2 ГБ на один объект.

📂 Структура решения (Solution Map)
Проект разделен на 5 модулей для обеспечения чистоты кода и удобства тестирования:
1. Slau.Common — Общие ресурсы
Библиотека, используемая и Мастером, и Воркерами.
Models/MatrixChunk.cs — контейнер для части строк матрицы, передаваемой узлу.
Networking/Message.cs — протокол обмена данными (команды управления и пакеты данных).
Serialization/FastBinarySerializer.cs — кастомная сериализация для быстрой передачи 50к элементов.
2. Slau.MathEngine — Математическое ядро
Библиотека с реализацией алгоритмов.
Algorithms/GaussJordanCore.cs — логика метода Гаусса-Джордана.
Algorithms/RelaxationSolver.cs — метод релаксации (локальный режим).
Storage/LargeMatrix.cs — обертка над памятью для работы с гигантскими матрицами.
3. Slau.Master — Главный узел (GUI)
Приложение администратора (WPF/WinForms).
Distribution/WorkerManager.cs — регистрация и мониторинг состояния вычислительных узлов.
Distribution/JobDispatcher.cs — алгоритм нарезки матрицы на части и балансировки нагрузки.
Logic/DistributedController.cs — главный цикл алгоритма (координация шагов Гаусса).
Views/MainWindow.xaml — интерфейс настройки, запуска и визуализации прогресса.
4. Slau.Worker — Вычислительный узел
Консольное приложение для расчетов.
Network/WorkerServer.cs — TCP-сервер, слушающий команды от Мастера.
Services/ComputationService.cs — выполнение математических операций над своей частью матрицы.
App.config — настройки лимитов (макс. RAM, количество ядер CPU).
5. Slau.Tests — Тестирование
Проект для верификации и апробации.
Unit/MathValidation.cs — тесты корректности на эталонных матрицах (3x3, 10x10).
Load/HeavyLoadTests.cs — нагрузочные тесты для замера времени при N=10000+.
